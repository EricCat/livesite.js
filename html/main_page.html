Welcome to LiveSite!

This is the default page that means that everything is working properly.

What to do now:

  • Create a new database where you want your livesite data to reside.
  • Create a config.js file in your livesite directory
  • Add database connection details to the config file

What to do then

  • You will find all themes in themes directory
  • You can add plugins to the plugin directory

You can not save your changes to this page, even though you are able to edit it. :)
